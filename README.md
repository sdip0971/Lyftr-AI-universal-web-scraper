# Universal Website Scraper MVP

## Setup & Run
1. Ensure Python 3.10+ and Node.js are installed.
2. Run:
   chmod +x run.sh
   ./run.sh
3. Open http://localhost:8000

## Test URLs
- https://en.wikipedia.org/wiki/Artificial_intelligence (Static)
- https://vercel.com/ (JS/Tabs)
- https://news.ycombinator.com/ (Pagination fallback)
